# 🚢 Titanic Survival Analysis | Data Analytics Project

## 📌 Project Summary

This project presents a comprehensive exploratory data analysis (EDA) of the Titanic Dataset to identify the key factors that influenced passenger survival.

The goal is not just to analyze the data, but to extract meaningful insights that demonstrate real-world analytical thinking using Python.

---

## 🎯 Objectives

* Understand survival patterns among passengers
* Identify important features affecting survival
* Perform structured data cleaning and preprocessing
* Communicate insights effectively using visualizations

---

## 📂 Dataset Details

The dataset contains demographic and travel-related information for Titanic passengers, including:

* Passenger demographics (Age, Gender)
* Socio-economic status (Passenger Class, Fare)
* Travel details (Embarked, Cabin)
* Survival outcome

---

## 🛠️ Tech Stack

* **Python**
* **Pandas** → Data cleaning & manipulation
* **NumPy** → Numerical computation
* **Matplotlib & Seaborn** → Data visualization
* **Jupyter Notebook** → Analysis environment

---

## 🔄 Project Workflow

### 1. Data Understanding

* Loaded dataset and explored structure
* Reviewed data types and summary statistics

### 2. Data Cleaning & Preprocessing

* Handled missing values (Age, Cabin, Embarked)
* Removed irrelevant or redundant columns
* Prepared dataset for analysis

### 3. Exploratory Data Analysis

* Univariate analysis for feature distribution
* Bivariate analysis with survival comparison
* Correlation analysis for numerical relationships

### 4. Data Visualization

* Survival distribution plots
* Gender and class-based comparisons
* Age and fare distribution analysis
* Correlation heatmap and boxplots

---

## 📊 Key Insights

* **Gender Impact:** Female passengers had a significantly higher survival rate
* **Class Influence:** First-class passengers showed better survival probability
* **Age Factor:** Children had relatively higher chances of survival
* **Fare Correlation:** Higher ticket fares were linked to increased survival
* **Embarkation Trends:** Survival varied across embarkation ports
